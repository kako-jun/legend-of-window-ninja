# legend-of-window-ninja

React + Vite + TypeScript で作成したアクションゲーム

Phaser 3 を使用したブラウザベースの忍者アクションゲームです。忍者として手裏剣を投げて敵を倒すゲームです。スマホでもPCでも快適にプレイできます。

## 操作方法

### PC

- 矢印キー: 忍者を移動
- スペースキー: 手裏剣を投げる

### スマホ・タブレット

- タップ: 移動と手裏剣投げ

## 技術スタック

- React 18
- Vite 6
- TypeScript 5
- Phaser 3
- Tailwind CSS 4
- Prettier + ESLint
- Husky + lint-staged

## 開発コマンド

```bash
# 依存関係のインストール
npm install

# 開発サーバーを起動
npm run dev

# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview

# コードのフォーマット
npm run format

# Lintチェック
npm run lint
```
