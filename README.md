# Legend of Window Ninja

ブラウザベースの忍者アクションゲーム。手裏剣を投げて敵を倒せ。

## 操作方法

### PC

- 矢印キー: 移動
- スペースキー: 手裏剣を投げる

### スマホ・タブレット

- タップ: 移動と手裏剣投げ

## セットアップ

```bash
git clone https://github.com/kako-jun/legend-of-window-ninja.git
cd legend-of-window-ninja
npm install
npm run dev
```

http://localhost:5173 で開く

## 技術スタック

- React 18 + TypeScript
- Vite 6
- Phaser 3
- Tailwind CSS 4
